## D. 요약 — 큰 그림

**이미 튼튼한 것(트레이너 코어):** OT/재등록 AI · 음성일지 자동작성 · 수업 차감 · 급여 · 라이브러리 · 인바디 · 운동 변화 추이. **회원앱 v1(열람 3종: 수업일지·인바디·무게추이)도 완료.**

**미완은 세 덩어리로 뭉침:**

1. **리포트·통계류** (트레이너 강약점 / 센터 강약점 / 월간·예측 매출 / 인바디 랭킹 / 팀 KPI 실데이터화) — 공통 토대 = **④ 클로징 통계 + 리포트 엔진.** 하나 세우면 여럿이 그 위에 얹힘.
2. **스케줄 자동화 에픽(#4)** — 트레이너 가능시간 → 회원 슬롯선택 → 관리자 파이프라인 → 배정(+추천). 뼈대(`appointment`·ScheduleBoard)는 있음.
3. **회원앱 자가입력 + 회원 AI** (식단·루틴·비포애프터·유산소·스케줄 체크) — **거의 다 프리미엄 회원 AI.** 그래서 **B1/B2 구독 + AI 게이트**가 선행이면 자연스럽게 열림.

**추천 순서(로드맵 정합):** 위생 정리 → **B1 구독 잠금** → **B2 프리미엄 게이트** → **④ 클로징 통계**(리포트류 토대) → **#4 스케줄 에픽** → **회원 자가입력 + 회원 AI(게이트)**. 미완 회원 기능 다수가 프리미엄 AI라, 수익화 스위치(B1/B2)를 먼저 까는 게 순서상 유리.
