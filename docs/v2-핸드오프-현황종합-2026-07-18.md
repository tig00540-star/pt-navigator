## 1. 프로젝트 개요

- **제품**: 오직 트레이너 (레포명 pt-navigator) — 퍼스널 트레이너용 세일즈·CRM·PT관리 웹앱(PWA).
- **브랜드**: "오직 트레이너", 레드 계열 브랜드 컬러.
- **사용자**: 트레이너(주), 원장(관리자), 회원(회원 앱).
- **스택**: Next.js 16 (App Router) · React 19 · 순수 JS/JSX(TS 아님) · Tailwind CSS v4(config 없음, 토큰은 `app/globals.css @theme`) · Supabase(Postgres + PostgREST + Auth + Storage) · Vercel · PWA.

---

## 2. 협업 워크플로우 (반드시 유지)

1. 웹Claude가 수정 스펙을 "클로드코드에 붙여넣을 수 있게" 작성.
2. 클로드코드가 로컬 레포에서 실제 코딩.
3. 대수가 `git diff`를 붙여넣음 → 웹Claude가 리뷰.
4. **커밋은 대수가 직접.** (웹Claude가 커밋 안 함)

---

## 3. 코드/UI 컨벤션 (절대 규칙)

- **색상은 토큰 클래스만 사용**: `primary`, `primary-soft`, `primary-strong`, `ink`, `sub`, `muted`, `line`, `bg`, `bg2/elevate`, `card`, `rose-600`, `amber-500`, `sky-500`.
- **동적 클래스 문자열 조립 금지** (Tailwind purge 때문) — 정적 삼항만 사용.
- **아이콘은 lucide-react만.**
- **새 문자열은 정적으로.**
- **표시 텍스트·색·간격·레이아웃만 변경. 로직은 절대 손대지 않음** — `ai_summary`, 회차 계산, save payload, RLS 불변.
- **git add는 파일 지정으로만.** `git add -A` / `git add .` 금지 (untracked인 `marketing/`, PDF들, `pt-navigator-landing.html`, 기타 문서가 커밋되면 안 됨).
- **SQL 마이그레이션은 Supabase SQL Editor에서 수동 실행.**
- 새 파일(신규 컴포넌트·마이그레이션 등)은 `git diff`에 안 뜨므로 **명시적 `git add` 필요**.

### 아키텍처 메모
- **mode-prop + children 슬롯 패턴**: 큰 컴포넌트를 view/record 슬라이스로 분리(로직 중복·순서 붕괴 없이).
- **memberStatus.js 순수 헬퍼**: `viewFor`, `activeContract`, `remainingSessions`, `reregisterDue`(paid<10), `reapproachToday`, `unclosedClosings`, `pastDueAppointments`, `toInactive`, `buildContract`, `initialStatus` 등. 분기는 `viewFor()`로만, status 직접 비교 금지.
- **AI 브리핑**: `/api/ot-brief` (phases: first/second/reregister/acute, Claude Sonnet). 추천 프로그램은 `pick_ref`로 트레이너 `pt_package`를 지목하고 **가격은 앱이 채움**(할루시네이션 방지). 출력은 반드시 한국어만.
- **RLS 주의**: `daily_workout_log`엔 UPDATE 정책 없음(무음 0-row 실패 위험). `.select()` 후 error null + 0 rows = 무음 실패 하드닝 필요. `member_photo` 삭제는 **스토리지 먼저 → 행 나중**(조인 정책이 풀리도록).

## 6. 제품 관련 확정된 판단 (재논의 방지용)
- 전화번호 뒷자리 중복 로그인: 3회 제한 **하지 않기로** 함.
- 재관찰 탭: 필요 없음(대수가 "잘못 본 것"이라 함).
- 1차 클로징 결과에도 보류/실패 라벨 적용(통합 `CLOSING_RESULT_OPTS`).
- OT 정보수정/PT 정보수정은 일부러 막혀있던 것 → 경고 모달과 함께 여는 것으로 결정.
- 1차 AI 지원이 세션에서만 보였던 이유 등은 설명 완료.
